# ⏱️ React Timer Game

A simple Timer Reaction Game built with [React](https://react.dev/ "React") where the goal is to stop the timer as close as possible to the target time.

The player starts a timer and must click Stop at the correct moment. If the player stops the timer exactly at the right time — they win 🎉. Otherwise, they lose ❌.

## 🎮 How the Game Works
1. The player clicks Start Timer.
2. The timer begins counting.
3. The player must click Stop at the correct moment.
4. If the timer is stopped exactly at the target time, the player wins.
5. If the timer is stopped too early or too late, the player loses.

## 📦 Installation
1. Clone the repository:
`git clone https://github.com/your-username/react-timer-game.git`

2. Go into the project folder:
`cd react-timer-game`

3. Install dependencies:
`npm install`

4. Start the development server:
`npm run dev`
